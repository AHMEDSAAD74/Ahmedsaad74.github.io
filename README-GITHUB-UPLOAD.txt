AHMED SAAD PORTFOLIO — GITHUB PAGES UPLOAD

FILES TO UPLOAD TO THE ROOT OF YOUR REPOSITORY:
- index.html
- Ahmed_Saad_CV.pdf

Do not upload the ZIP file itself into GitHub.

STEPS:
1) Extract this ZIP file on your computer.
2) Open your GitHub repository.
3) Click Add file > Upload files.
4) Drag BOTH files (index.html and Ahmed_Saad_CV.pdf) into the upload area.
5) Scroll down and click Commit changes.
6) Go to Settings > Pages.
7) Under Build and deployment, choose:
   Source: Deploy from a branch
   Branch: main
   Folder: / (root)
8) Click Save.
9) Wait a few minutes (GitHub says publishing can take up to 10 minutes), then refresh the Pages settings page. GitHub displays the site URL there.

IMPORTANT:
- If your repository name is username.github.io, the final URL is https://username.github.io
- If it has any other name, the URL is usually https://username.github.io/repository-name/
- To update the CV later, upload a replacement file with the exact name: Ahmed_Saad_CV.pdf
